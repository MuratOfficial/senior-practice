      def merge_intervals(intervals: list[list[int]]) -> list[list[int]]:
          if not intervals:
              return []
          ordered = sorted(intervals, key=lambda pair: pair[0])
          merged = [ordered[0][:]]
          for start, end in ordered[1:]:
              last = merged[-1]
              if start <= last[1]:
                  last[1] = max(last[1], end)
              else:
                  merged.append([start, end])
          return merged

Реализуйте `merge_intervals(intervals)` — слияние всех пересекающихся интервалов (LeetCode 56). Одна из самых частых задач на массивы; проверяет ключевую идею «сначала отсортировать, потом жадно пройти».

Требования:

- пересекающиеся и **соприкасающиеся** (`[1,4]` и `[4,5]`) интервалы сливаются;
- вход может быть неотсортирован;
- результат — непересекающиеся интервалы по возрастанию.

<!-- explanation -->

Инсайт, ради которого задача существует: **после сортировки по левой границе** все интервалы, которые нужно слить, идут подряд. Это превращает задачу из «сравнить каждый с каждым» (O(n²)) в один линейный проход после сортировки — итог **O(n log n)**, где доминирует сортировка.

Проход жадный: держим `merged` с последним «накопленным» интервалом. Для очередного `[start, end]`:

- `start <= last[1]` → пересекается (или касается) → расширяем правую границу: `last[1] = max(last[1], end)`. `max` обязателен — иначе вложенный интервал `[2,3]` внутри `[1,10]` ошибочно ужмёт границу до 3.
- иначе → разрыв → добавляем новый интервал.

Два подвоха:

1. **Соприкосновение считается пересечением** (`<=`, не `<`): `[1,4]` и `[4,5]` дают `[1,5]`. Если по условию нужно иначе — меняем на строгое `<`.
2. **Копия первого интервала** (`ordered[0][:]`) — мы мутируем `last[1]` на месте; без копии испортили бы исходные данные вызывающего.

**Follow-up:** «вставить интервал в уже отсортированный список и слить» (LeetCode 57 — без полной сортировки, за O(n)); «найти максимальное число одновременных пересечений» (sweep line с событиями +1/−1); та же идея на брони переговорок (minimum meeting rooms).
